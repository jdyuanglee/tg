I created a responsive CMS website from scratch using PHP with WordPress, which includes designing, wireframing, coding the static page and then transfer it into dynamic page . There are 5 template pages in total except for the front page and the 404 page. Those five are sidebar page, post listing page, post-detail page, general template page and search (result) page. Additionally, I implemented some wordpress plugins like Advanced Custom Fields for advanced customization without changing the design.

Static Page

After weeks of researching, brainstorming, sketching and wireframing, I started coding the static page for the website using plain HTML/CSS and JavaScript/ JQuery for the micro-interactions. The page was fully polished as a static website in March, 2018 for my final project of the web dev class. Page Link

Dynamic Page

Starting from June of 2018, I was taking the class of using online content management system (WordPress) and designing the client dashboard that they can easily adding, editing the page, changing the customized nav menu and profile picture.

We use PHP to remote WordPress backend system and managing the front end files. By the end of the term, I have six templates built up, which are Home, General, Post Listing, Post Detail, Search Result and 404 page. Beside that, I made the dynamic navigation, categories system with achieve page and Advanced Customer Fields for setting rules for the inputs.
