Recipes cookbook is a project I created for scripting II class, which is about learning PHP and MySQL. We first built up a static web page for Alpha version using only HTML, CSS and JS. After then, we started included PHP into HTML, setting up database connection, query passing and searching features. We set up the database by designing, planning and filling the information into a CSV forms then import to MySQL.

There are over 60 recipes in total on the cookbook and each of them have different images, descriptions, ingredients list and cooking procedures. Users can use the search function and filter to navigate to their recipe easily.
