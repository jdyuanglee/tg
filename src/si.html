For the final project of user interface design class, I redesigned the interface of si.com and created a responsive prototype based on real contents. My approaches of the process is using mobile first sequence. I did a lot of researhes on the relative websites of Sport Illustrated like ESPN, FOX, Bleacher Report and mapped down the features and analyzed the hierarchy.

I used UXPin as my prototyping tool, which is also the first tool I had even tried. I added real content to the prototype as well as the real ads. Interactions, transitions and animations added made the prototype well-polished. At the end, I also included the night shift that users can choose to view the website on dark mode or day mode.
