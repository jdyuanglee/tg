The process of designing my person branding is consist of creating a unique logo, building a fully-polished website and a solid resume. I started the whole process of building in September and still keep polishing it. I used the knowledge of graphic design, UI designer, content managing as well as designing the whole experience by adding animation, adjusting layout, etc.

Firstly, I designed the logo on one of my graphic design project, which is about learning typeface. I made the composition using the letter t and g and then exploring the combination by playing around their outlines and then came out the final idea.

After that, I created a website from scratch starting by making the wireframe. I did usability testings and asked for feedbacks along with building up the wireframe. Once the design concept is fully set up, I started coding the website using pain HTML, CSS and JS.

Lastly, I created a creative resume and refined the content. Throughout the process, I reevaluated my website and the style guide and then created a template using Adobe InDesgin.
