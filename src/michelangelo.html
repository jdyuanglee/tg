The project is for Workflow Optimization class, learning collaborative workflow techniques and maintaining deliverable schedule. The theory of coloring book itself is about using the grouping in svg and Jquery to detect hover, click for selecting and filling color.

For the workflow optimization, we learnt and applied those techniques to our building process. We coded bash scripting, gulp and used npm modules to handle those jobs. Moreover, we used and forked GitHub from our project manager and work collaboratively over there by using features branches and merge regulations.
